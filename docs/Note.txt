=== Chuẩn bị dữ liệu YOLO `scripts/prepare_yolo_dataset.py` ===
# Các tham số:
--data-dir: Thư mục chứa dữ liệu gốc (mặc định: ./data)
--output-dir: Thư mục để lưu dataset chuẩn YOLO (mặc định: ./dataset)
--frame-interval: Khoảng cách giữa các frame được trích xuất (mặc định: 10)
--val-split: Tỷ lệ dữ liệu dành cho validation (mặc định: 0.2)
--splits: Danh sách các split cần xử lý (mặc định: ['train'])
--single-match: Chỉ xử lý một thư mục match cụ thể (tùy chọn)

# Thử nghiệm với một match
python scripts/prepare_yolo_dataset.py --single-match football_train/Match_1824_1_0_subclip_3 --frame-interval 10 --val-split 0.2

# Xử lý tất cả matches trong tập train
python scripts/prepare_yolo_dataset.py --splits train --frame-interval 10 --val-split 0.2

# Xử lý tất cả matches trong tập test
python scripts/prepare_yolo_dataset.py --splits test --frame-interval 10 --val-split 0.0

# Xử lý cả train và test
python scripts/prepare_yolo_dataset.py --splits train test --frame-interval 10 --val-split 0.2
